# todo_app_frontend

React 17 Todo app frontend (no backend).

## Features
- Create tasks
- Inline edit (double-click or Edit button)
- Toggle complete/incomplete
- Delete tasks
- Filter: All / Active / Completed
- Local persistence via `localStorage`

## Run
```bash
npm install
npm start
```

The app persists todos in the browser under the key:
`smart_todo_list.todos.v1`
